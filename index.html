Instruções

Olá usuário. Este sistema possui duas formas para realizar a tarefa de cálculo de quota a ser pago por cada participante da lista.

  • Método 1: Essa opção é aceita listas separadas contento o nome e e-mail e a outra possuindo o produto, quantidade e o preço unitário. Para facilitar deixei dois arquivos para download abaixo como modelo.
    • Nomes: modelo 1 - nomes
    • produtos: modelo 1 - produtos
  • Método 2: Caso decida optar em unificar as listagens num só arquivo utilize está opção que é capaz de ler pastas existentes nos documentos do Excel. Caso esteja em dúvida use o nosso modelo disponível no link abaixo.
    • Nomes e produtos: modelo 2 - nomes e produtos
  • Observações: Está aplicação não suporta arquivos CSV. Por favor, utilize documentos xlsx ou xls.
